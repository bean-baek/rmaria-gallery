## Verification Protocol

1. `npm install` completes without errors (warn the user about sharp's native build on Windows if it surfaces — recommend Node ≥ 18 LTS).
2. `npm run dev` boots, opens at `http://localhost:5173`, renders an empty page with **zero console errors**.
